*Compile: make
*Run: ./aes <key> <NumSampels>
The `<key>` is a 32-character hexadecimal number representing the AES key used for encryption, and `<NumSamples>` indicates the number of samples to generate.
The code generates `<NumSamples>` and saves them into the `aes_output.txt` file.
